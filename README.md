## Muts Website Server

Server/backend program which handles user's request from 
[muts_website_client](https://github.com/talukanyani/muts_website_client).
It validates user's data, stores to database, and sends email to admin.

### Technologies Used

- JavaScript
- Node.js
- Express
- Firestore
